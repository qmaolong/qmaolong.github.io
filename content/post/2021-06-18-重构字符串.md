## 问题：

给定一个字符串 s ，检查是否能重新排布其中的字母，使得两相邻的字符不同。
返回 s 的任意可能的重新排列。若不可行，返回空字符串 "" 。

#### 示例 1:
```
输入: s = "aab"
输出: "aba"
```

#### 示例 2:
```
输入: s = "aaab"
输出: ""
```

#### 提示：
- 1 <= s.length <= 500
- s 只包含小写字母

---

## 解题：

```golang
package main

/*
 * @lc app=leetcode.cn id=767 lang=golang
 *
 * [767] 重构字符串
 */

// @lc code=start
func reorganizeString(s string) string {
	countMap := make(map[rune]int)
	for _, char := range s {
		countMap[char] += 1
		println(string(char))
	}
	return s
}

// @lc code=end

```
